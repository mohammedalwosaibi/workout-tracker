# Workout Tracker

## Key Rules
- **Always ask clarifying questions before implementing any prompt.** Do not start coding without understanding the user's intent.

## Tech Stack
- Next.js (App Router) with TypeScript
- Tailwind CSS for styling
- localStorage for client-side data persistence
- Playwright for testing

## Project Structure
- `src/app/` — Next.js App Router pages
- `src/components/` — Reusable components (ui/, layout/, domain-specific/)
- `src/hooks/` — Custom React hooks (useLocalStorage, useExercises, etc.)
- `src/lib/` — Utilities and pure functions (storage, progressive-overload)
- `src/types/` — TypeScript type definitions
- `tests/` — Playwright test files
